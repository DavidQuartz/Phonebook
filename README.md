# A Phonebook App

## A WeSkillz React Task

This app is a simple phone book app built with create react app.

### Stack:

    1. React
    2. Context API
    3. Bootstrap
    4. React-Bootstrap
    5. Firebase
    6. Node-Sass

### Features:

    1. Login
    2. Create Contact
    3. View all Contacts
    4. View one contact
    5. Edit Contact
    6. Delete Contact
